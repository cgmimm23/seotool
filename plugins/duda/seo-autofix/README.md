# SEO AutoFix for Duda

AI-powered SEO auto-fixer for Duda websites. Connects to your AI SEO dashboard and automatically applies fixes.

## Setup

1. Get your API key from seo.cgmimm.com → Settings → API Keys
2. Get your Site ID from the SEO dashboard
3. Get your Duda API credentials from Duda → Settings → API Access
4. Install this app on your Duda site
5. Enter all credentials in the settings
6. Click "Sync & Apply Fixes"

## What it fixes

- Meta titles and descriptions
- Open Graph tags
- Schema/JSON-LD markup
- Image alt text
- Canonical URLs
