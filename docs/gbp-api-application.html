Product Overview

AI SEO by CGMIMM is a full-stack SEO platform for small and mid-size businesses. It combines automated technical audits, rank tracking, page optimization, and local-presence management in a single dashboard. Every feature is powered by AI — the platform interprets raw data and writes a plain-English action plan the business owner can execute without SEO expertise.

Customers subscribe at Starter ($59.95/mo), Pro ($149/mo), or Enterprise (custom) tiers and use the tool to manage SEO for their own websites. Agencies on Pro and Enterprise manage multiple client sites.

Why We Need Business Profile API Access

Local search visibility is a core part of SEO for the business owners we serve (contractors, restaurants, medical practices, retail, professional services). A correctly-configured, fully-optimized Google Business Profile is one of the single highest-impact local-ranking factors for our customers.

Our users are already Google Business Profile owners (or are creating their first listing). They grant our application explicit OAuth consent to manage their own profiles on their behalf.

Specific Use Cases

1. Business Profile Creation & Claiming

  • Guided setup wizard that collects NAP (name, address, phone), categories, hours, service areas, and description
  • AI-generated, keyword-optimized business descriptions that follow Google's 750-character limit and content guidelines
  • Submits the completed listing via the Business Information API under the user's own Google Business Profile Manager account
  • Walks the user through Google's verification flow (postcard, phone, email, video) via the Verifications API

2. Profile Sync & Auditing

  • Pulls the user's existing GBP locations and audits completeness (missing categories, photos, service areas, attributes)
  • Flags gaps and opportunities in the same plain-English action list the rest of the tool produces
  • Keeps NAP data consistent between the user's website, GBP, and other citation sources we push to

3. Review Management

  • Surfaces Google reviews across all of a user's locations in a single inbox
  • AI suggests professional, brand-appropriate responses to reviews (user approves before posting)
  • Alerts users to new negative reviews so they can respond quickly

Data Handling & Security

  • OAuth consent: Users authorize our app via Google's standard OAuth 2.0 consent flow with the minimum scopes required (business.manage for GBP, plus read-only Search Console and Analytics where applicable).
  • Token storage: Access and refresh tokens are stored encrypted in our Supabase Postgres database (AES-256 at rest, TLS in transit). Tokens are associated only with the user who granted them and are never shared across accounts.
  • Row-level security: Every database table uses Postgres row-level security so users can only read and write their own data.
  • Write operations: All writes to Business Profile data (creating listings, updating info, posting review replies) are user-initiated from the UI. We do not perform unattended background writes to Google.
  • Disconnection: Users can revoke access at any time from their settings page; we immediately delete the stored tokens.
